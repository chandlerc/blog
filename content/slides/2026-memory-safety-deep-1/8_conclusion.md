# Conclusion

---

## In summary

Smooth and incremental transition from C++ to safe Carbon

- Memory Safe
  - Compile-time use-after-free checking
- Expressive
  - Common C++ patterns made safe without rearchitecting
- Incremental
  - Interop and permissive mode
